# Vue Sample for Neuron Dashboard Integration

This sample targets Neuron dashboard style (Vue + Element UI style components).

## Files

- `RemoteControl.vue`: full page sample for remote control form/actions/status
- `remoteControlApi.js`: API client for `/api/v2/remote/*`
- `integration-steps-neuron-dashboard.md`: route/menu wiring steps for `neuron-dashboard` source repo

## Why this folder exists

The current repository contains only `neuron-dashboard.zip` static build artifacts, not full dashboard source.
So direct route/menu source edits cannot be applied here.
This folder provides copy-ready source snippets for the actual dashboard source repository.
